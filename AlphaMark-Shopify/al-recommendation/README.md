# AlphaMark Shopify App

Shopify app wrapper for the AlphaMark AI recommendation system.

This app contains the Shopify configuration and theme app extension that loads the Railway-hosted AlphaMark widget on a Shopify storefront.

## Useful Commands

```sh
shopify app dev
shopify app deploy
```

The backend API lives in the parent AlphaMark backend project and is deployed separately to Railway.
